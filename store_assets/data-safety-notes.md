# QRCanvas Data Safety Notes

- Data collected: None.
- Data shared: None.
- Account creation: Not available and not required.
- Ads: None.
- Tracking: None.
- Internet connection: Not required for app functionality.
- QR content: Entered by the user and stored locally only when the user saves a project.
- Saved projects: Stored locally on the device with AsyncStorage.
- Exported files: PNG and JSON exports are created on-device and shared only when the user chooses a destination.
- Wi-Fi QR codes: May contain network names and passwords entered by the user. They remain local unless exported or shared by the user.
- Location QR codes: Created from manually entered coordinates or pasted map URLs. The app does not request location permission.
- Data deletion: Users can delete individual projects or reset all QRCanvas local data in Settings.
- Permissions: No camera, contacts, location, internet, or external storage permission is required by the app.
